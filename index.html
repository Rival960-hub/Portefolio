Veille Technologique

Thème choisi : L’évolution des frameworks JavaScript

En tant que futur développeur, il est essentiel de suivre les évolutions des outils utilisés dans le développement web. Les frameworks JavaScript comme React, Vue.js et Angular permettent de créer des applications dynamiques et modernes.

Outils utilisés pour la veille

  • Google Alerts (mots-clés : “JavaScript framework”, “React”, “Vue.js”)
  • Sites spécialisés : OpenClassrooms, MDN Web Docs, Journal du Net
  • Communautés : GitHub, Stack Overflow

Ressources

  • Documentation officielle de React
  • Documentation officielle de Vue.js
  • Documentation officielle de Angular

Bilan

Cette veille m’a permis de comprendre que chaque framework a ses avantages : React est très populaire et flexible, Vue.js est simple à prendre en main, et Angular est complet mais plus complexe. En tant qu’étudiant, je retiens qu’il est important de tester plusieurs outils pour développer mes compétences.
